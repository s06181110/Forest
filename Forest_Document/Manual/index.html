インストール方法

1. Forest(GitHub)にアクセスし、中央右のClone or downloadをで(a)cloneか(b)download zipを選択する。

以下はディレクトリを作成するが、任意のディレクトリに持ってきたい場合は、

$ mkdir -p ~/SE_long_term_hw

の部分を

$ cd (任意のディレクトリ)

に書き換えてコマンドを実行してください。

2-a. Gitからcloneする場合、ディレクトリを作成し、以下のコマンドを実行する。

$ mkdir -p ~/SE_long_term_hw ; git clone https://github.com/s06181110/Forest

2-b. zipを解凍する場合、ディレクトリを作成し、以下のコマンドを実行する。

$ mkdir -p ~/SE_long_term_hw ; unzip Forest-master.zip

3. clone又はdownloadしたディレクトリへ移動する。

$ cd ~/SE_long_term_hw

又は

$ cd (任意のディレクトリ)

4. 中身の確認。

ls -l

5. makeする。

$ make
